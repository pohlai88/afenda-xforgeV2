## DELETE /plants/`{id}`

Deletes a plant by ID.

### Parameters

| Name | In   | Type   | Description    |
| ---- | ---- | ------ | -------------- |
| id   | path | string | Plant identifier |

### Response

```json title="204 No Content"
{}
```

### Example

```bash title="curl"
curl -X DELETE https://api.example.com/plants/1 \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```
