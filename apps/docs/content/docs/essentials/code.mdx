## Basic

### Inline Code

To denote a `word` or `phrase` as code, enclose it in backticks.

```
To denote a `word` or `phrase` as code, enclose it in backticks.
```

### Code Block

Use fenced code blocks with a language identifier for syntax highlighting. In Fumadocs, add an optional title with `title="..."`:

```java title="HelloWorld.java"
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```


## Highlighting lines

Highlight a line with an inline comment:

```ts
console.log("Highlighted"); // [!code highlight]
```

Highlight a word:

```ts
console.log("Highlighted"); // [!code word:Highlighted]
```
