## POST /plants

Creates a new plant.

### Request body

```json
{
  "name": "Monstera"
}
```

### Response

```json title="201 Created"
{
  "id": "2",
  "name": "Monstera"
}
```

### Example

```bash title="curl"
curl -X POST https://api.example.com/plants \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name":"Monstera"}'
```
