## Welcome

Document API endpoints with MDX pages under `content/docs/api-reference/`. You can describe request/response shapes, authentication, and examples in markdown and code blocks.

The starter includes example endpoints for a fictional Plant Store API. The OpenAPI spec from the Mintlify starter is kept at `api-reference/openapi.json` for reference.

## Authentication

Example bearer token authentication:

```json
"security": [
  {
    "bearerAuth": []
  }
]
```

Requests should include:

```http
Authorization: Bearer YOUR_API_TOKEN
```
